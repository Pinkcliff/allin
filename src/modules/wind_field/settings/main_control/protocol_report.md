## 一、修改内容总结

将 `udp_fan_sender.py` 的协议从旧版改为与 `fan_control.py` 完全一致的格式：

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| 帧号字节序 | 大端序 | **小端序** `struct.pack('<I')` |
| 头部电驱板ID | 写实际板号(0-9) | **固定 0x00** |
| 数据长度字节序 | — | **大端序** `struct.pack('>H', 640)` |
| 数据区布局 | 单板数据从字节10开始 | **10板打包，板N偏移=N×64** |
| 每风扇占 | 2字节(1个参数) | **4字节(2个参数副本)** |
| PWM字节序 | 大端序 | **小端序** `struct.pack_into('<H')` |
| PWM满值 | 25600 | **1000** |
| 发送粒度 | 每板1包(100包) | **每链路1包(10包)** |
