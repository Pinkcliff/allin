# 动态曲面分析工具使用说明

本工具用于分析和可视化形如 `z = f(x, y, t)` 的动态曲面函数，特别适合研究曲面上固定点处 z 值随时间的变化。

## 功能特点

1. **3D曲面可视化**：显示任意时刻的曲面形状
2. **时间序列分析**：提取并展示固定坐标点上 z 值随时间的变化
3. **多点对比**：同时分析多个点的时间演化
4. **时空热力图**：展示 z 值在时空中的分布
5. **动画演示**：创建曲面随时间变化的动画
6. **数据导出**：将分析结果导出为 CSV 文件

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 快速开始（演示模式）

```bash
cd src
python main.py --mode demo
```

这将运行一个演示，展示双波源干涉图样的分析。

### 2. 交互式模式

```bash
cd src
python main.py
```

按提示操作：
1. 选择预定义函数或选择自定义函数
2. 输入 x、y、t 的范围
3. 输入要分析的固定点坐标

### 3. 使用自己的函数

编辑 `config.py` 文件中的 `your_function` 函数：

```python
def your_function(x, y, t):
    """
    在这里定义您的动态曲面函数
    x, y, t 可以是标量或numpy数组
    返回 z 值
    """
    # 示例：三维驻波
    return np.sin(x) * np.sin(y) * np.cos(2*t)
```

### 4. 直接使用类

```python
from dynamic_surface import DynamicSurface
from point_analyzer import PointAnalyzer

# 定义您的函数
def my_surface(x, y, t):
    return np.sin(x + t) * np.cos(y - t)

# 创建动态曲面实例
ds = DynamicSurface(my_surface)

# 绘制 t=2 时刻的曲面
ds.plot_surface_at_time((-5, 5), (-5, 5), t=2)

# 分析固定点
analyzer = PointAnalyzer(my_surface)
analyzer.add_point(1, 1, "点A")
analyzer.add_point(-1, -1, "点B")
analyzer.plot_multiple_points((0, 10))
```

## 示例函数

工具内置了多个示例函数：

1. **简单波动** (`simple_wave`): `z = sin(x + t) * cos(y - t)`
2. **径向波动** (`radial_wave`): 从中心向外传播的波
3. **高斯波包** (`gaussian_packet`): 在平面上移动的高斯波包
4. **驻波** (`standing_wave`): 二维驻波模式
5. **螺旋波** (`spiral_wave`): 旋转传播的螺旋波
6. **干涉图样** (`interference`): 两个点源的干涉
7. **孤立子** (`soliton`): 二维孤立子解

## 输出说明

1. **3D曲面图**：显示特定时刻的曲面形状
2. **时间序列图**：显示固定点 z 值随时间的变化曲线
3. **时空热力图**：显示不同时刻 z 值的空间分布
4. **复合图**：同时显示 3D 曲面和时间序列
5. **CSV 文件**：包含所有分析点的数值数据

## 常见应用场景

1. **波动物理学**：分析波的传播、干涉、衍射
2. **热传导**：温度分布随时间的变化
3. **流体动力学**：液面波动分析
4. **量子力学**：波函数的时间演化
5. **信号处理**：二维信号的时间变化

## 注意事项

1. 确保函数返回的是数值类型（不是复数）
2. 对于复杂函数，可能需要调整分辨率参数
3. 动画生成可能需要较长时间，建议先用静态图调试
4. 导出的数据可以使用 Excel 或其他工具进一步分析
