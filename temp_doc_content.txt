二、软件概述
2.1 软件背景
本软件是一款面向微小型无人机风洞测试的专业综合评估系统，旨在解决无人机风场环境测试中的多设备协同控制、数据采集与分析、可视化监控等关键问题。系统集成了风扇阵列控制、环境参数监测、动作捕捉数据采集、CFD前处理等多个功能模块，为无人机风洞测试提供一体化解决方案。
2.2 应用领域
• 无人机风洞试验测试
• 空气动力学研究
• 风场环境模拟与评估
• 多传感器数据融合分析
• 实验设备集成控制
2.3 主要用途
1. 风扇阵列控制：配置1600台风扇的静态转速参数
2. 环境参数监测：实时采集温度、湿度、风速、压力等传感器数据
3. 动作捕捉集成：接收和处理动捕系统数据，可视化无人机姿态轨迹
4. 数据管理分析：提供历史数据查询、统计分析、数据导出功能
5. CFD前处理：生成风场测试场景配置文件
6. 系统监控：统一监控各子系统运行状态

三、功能说明
3.1 系统架构
本软件采用模块化设计，主要包含以下子系统：
• 融合平台主入口
• 全局监控仪表盘
• 风场编辑器
• 硬件控制模块
• 传感器数据采集模块
• CFD前处理模块
• 动捕数据可视化模块
• 核心数据层（SQLite/BerkeleyDB）
3.2 核心功能模块
3.2.1 全局监控仪表盘
• 系统监控Dock：显示CPU、内存、网络等系统资源使用情况
• 通讯状态Dock：监控各通讯协议的连接状态（TCP/IP、EtherCAT、Modbus等）
• 环境监测Dock：实时显示温度、湿度、压力、风速等环境参数
• 日志中心Dock：系统运行日志的实时显示和过滤
• 动捕数据Dock：显示动作捕捉系统的实时数据
• 风机状态Dock：1600台风扇的转速设置查看和热力图显示
3.2.2 风场编辑器
• 3D可视化编辑：在3D场景中布置风速区域和目标点
• 表达式解析：支持自定义数学表达式定义风速分布
• 模板库管理：预设多种常用风场模式（层流、湍流、梯度风等）
• 时间轴控制：支持风场随时间变化的动画编辑
• 实时预览：实时查看风场矢量和风速分布
3.2.3 硬件控制模块
实现对各类硬件设备的统一控制：
• 风扇阵列控制：
  - 支持1600台风扇（40×40阵列）的静态转速设置
  - 提供批量控制、区域控制、梯度控制等多种控制模式
  - 支持CSV格式控制序列导入和回放
• Modbus设备控制：
  - 支持俯仰伺服、造雨装置、喷雾装置等设备
  - 标准Modbus RTU/TCP协议
• EtherCAT设备控制：
  - 支持电驱、风速传感器、温度传感器等
  - 微秒级实时控制周期
3.2.4 传感器数据采集
提供完整的传感器数据采集和管理功能：
• 1600路 风扇转速设置
（具体数值取决于硬件配置和使用环境）
（该指标为系统设计参考值，实际运行中可能有所不同）
• 64路 温度传感器（-20~80℃）
• 64路 风速传感器（0~30m/s）
• 4路 温湿度传感器
• 1路 大气压力传感器

数据存储与管理：
• SQLite实时数据缓存
• BerkeleyDB历史数据归档
• CSV格式导出
• 历史数据查询和图表显示
• 统计分析（最大值、最小值、平均值、标准差等）
3.2.5 CFD前处理模块
• 场景生成：根据风场配置生成CFD场景文件
• 网格生成：自动生成计算域网格
• 边界条件设置：自动设置入口、出口、壁面等边界条件
• 风扇模型映射：将风扇阵列映射到CFD边界条件
3.2.6 动捕数据可视化
• CSV数据加载：支持LuMo动捕系统导出的CSV格式
• 3D轨迹显示：标记点的3D空间轨迹可视化
• 播放控制：支持播放、暂停、停止、跳转功能
• 统计分析：每个标记点的位置统计分析
• 数据导出：支持导出处理后的数据

四、技术特点
4.1 技术架构
• 开发语言：Python 3.8+
• GUI框架：PySide6 (Qt6)
• 数据可视化：Matplotlib, PyQtGraph
• 数据存储：SQLite, BerkeleyDB
• 通讯协议：Socket, Modbus-tk, EtherCAT

4.2 关键技术
• 多线程架构：采用工作线程模式，保证UI响应流畅

• 数据缓存策略：SQLite实时缓存 + BerkeleyDB持久化的双层存储
• 表达式解析引擎：支持复杂数学表达式的动态解析和计算
• 3D可视化技术：基于OpenGL的高性能3D渲染

4.3 性能指标

4.4 创新点
1. 一体化融合平台：首次将风场控制、数据采集、动捕、CFD等功能集成于统一平台
2. 大规模阵列控制：实现1600台风扇的静态配置管理
3. 自定义风场表达式：支持用户通过数学表达式定义任意风场分布
4. 多源数据融合：整合传感器、动捕等多源数据

二、安装与配置
2.1 系统要求
硬件要求：
• CPU: Intel i5 或同等性能处理器
• 内存: 8GB 以上（推荐 16GB）
• 硬盘: 100GB 可用空间

2.2 安装步骤
1. 安装Python环境
   下载并安装 Python 3.8+，确保 pip 可用

2. 安装依赖库
   pip install PySide6 sqlite pyBERKELEY matplotlib numpy pandas pyqtgraph

3. 配置系统参数
   编辑 config.py 文件，设置通讯参数、数据库连接等

三、快速开始
3.1 启动系统
双击运行 main.py 或在命令行执行：
python main.py

系统将显示启动器界面，提供三个选项：
1. 全局监控仪表盘 - 完整的监控界面
2. 融合系统 - 仪表盘 + 传感器数据

四、功能模块使用说明
4.1 全局监控仪表盘
• 系统监控：显示CPU、内存、网络使用情况
• 通讯状态：显示各通讯协议连接状态
• 环境监测：显示温度、湿度、风速、压力等参数
• 日志中心：查看系统运行日志
• 动捕数据：显示动作捕捉实时数据
• 风机状态：显示1600台风扇转速值

4.2 风场编辑器
新建风场配置：
1. 点击"新建"按钮
2. 设置风场参数（名称、描述）
3. 在3D视图中添加风速区域

添加风速区域：
1. 选择"添加区域"工具
2. 在3D场景中点击确定区域位置
3. 设置区域参数（位置、尺寸、风速表达式）

保存和导出：
• 保存项目：保存为 .wf 格式
• 导出CSV：导出风速分布数据
• 导出控制序列：导出到硬件控制系统

4.3 硬件控制
连接设备：
1. 打开硬件控制面板
2. 配置通讯参数
3. 点击"连接"按钮

风扇控制：
• 单个设置：选择风扇，设置转速值，点击应用
• 区域控制：选择区域，设置转速值，点击应用
• 梯度控制：设置起终点转速值，选择方向，点击应用

4.4 传感器数据采集
创建采集任务：
1. 打开传感器数据采集窗口
2. 设置采集参数（采样率、时长）
3. 点击"开始采集"

查看历史数据：
1. 切换到"数据查看"标签页
2. 选择要查看的采集记录
3. 查看数据详情和统计图表

五、常见问题
问题1：系统无法启动
解决方案：
1. 检查Python是否正确安装
2. 检查依赖库是否安装完整
3. 在命令行运行查看错误信息

问题2：无法连接硬件设备
解决方案：
1. 检查网络连接
2. 确认设备IP地址配置正确
3. 检查防火墙设置
4. 确认设备已上电

问题3：数据采集失败
解决方案：
1. 检查SQLite服务是否启动
2. 确认传感器连接正常
3. 检查磁盘空间是否充足

    result = []

        if c.isupper():

def calculate_median(values):
    """计算中位数"""
    if not values:
        return 0
    sorted_values = sorted(values)
    n = len(sorted_values)
    if n % 2 == 0:
        return (sorted_values[n // 2 - 1] + sorted_values[n // 2]) / 2
    return sorted_values[n // 2]

def calculate_stddev(values):
    """计算标准差"""
    if len(values) < 2:
        return 0
    avg = sum(values) / len(values)
    variance = sum((x - avg) ** 2 for x in values) / len(values)
    return variance ** 0.5

def find_percentile(values, percentile):
    """计算百分位数"""
    if not values:
        return 0
    sorted_values = sorted(values)
    k = (len(sorted_values) - 1) * percentile / 100
    f = math.floor(k)
    c = math.ceil(k)
    if f == c:
        return sorted_values[int(k)]
    d0 = sorted_values[int(f)] * (c - k)
    d1 = sorted_values[int(c)] * (k - f)
    return d0 + d1

        self.max_val = max_val
    
    def is_in_range(self, value):
        """检查值是否在范围内"""

def compare_versions(v1, v2):
    """比较版本号"""
    parts1 = [int(x) for x in v1.split('.')]
    parts2 = [int(x) for x in v2.split('.')]
    for p1, p2 in zip(parts1, parts2):
        if p1 > p2:
            return 1
        elif p1 < p2:
            return -1
    if len(parts1) > len(parts2):
        return 1
    elif len(parts1) < len(parts2):
        return -1
    return 0

def rotate_array(arr, n):
    """旋转数组"""
    if not arr:
        return arr
    n = n % len(arr)
    return arr[-n:] + arr[:-n]

def merge_sorted_arrays(arr1, arr2):
    """合并两个有序数组"""
    result = []
    i = j = 0
    while i < len(arr1) and j < len(arr2):
        if arr1[i] <= arr2[j]:
            result.append(arr1[i])
            i += 1
        else:
            result.append(arr2[j])
            j += 1
    result.extend(arr1[i:])
    result.extend(arr2[j:])
    return result

def build_url(base, path, params=None):
    """构建URL"""
    url = base.rstrip('/') + '/' + path.lstrip('/')
    if params:
        query = '&'.join(f'{k}={v}' for k, v in params.items())
        url += '?' + query
    return url

    """解析查询字符串"""
    result = {}
    for pair in query.split('&'):
        if '=' in pair:
            k, v = pair.split('=', 1)
            result[k] = v
    return result

def bubble_sort(arr):
    """冒泡排序"""
    result = arr.copy()
    n = len(result)
    for i in range(n):
        for j in range(n - i - 1):
            if result[j] > result[j + 1]:
                result[j], result[j + 1] = result[j + 1], result[j]
    return result

def chunked(items, size):
    """分块迭代"""
    for i in range(0, len(items), size):
        yield items[i:i + size]

def first(items):
    """获取第一个元素"""
    return items[0] if items else None

def last(items):
    """获取最后一个元素"""
    return items[-1] if items else None

def take(items, n):
    """获取前n个元素"""
    return items[:n]

def skip(items, n):
    """跳过前n个元素"""
    return items[n:]

# ==================== 字典工具 ====================
def merge_dicts(*dicts):
    """合并多个字典"""
    result = {}
    for d in dicts:
        result.update(d)
    return result

def get_nested_value(d, keys, default=None):
    """获取嵌套字典的值"""
    for key in keys:
        if isinstance(d, dict) and key in d:
            d = d[key]
        else:
            return default
    return d

def set_nested_value(d, keys, value):
    """设置嵌套字典的值"""
    for key in keys[:-1]:
        if key not in d:
            d[key] = {}
        d = d[key]
    d[keys[-1]] = value

# ==================== HTTP相关 ====================
def build_url(base, path, params=None):
    """构建URL"""
    url = base.rstrip('/') + '/' + path.lstrip('/')
    if params:
        query = '&'.join(f'{k}={v}' for k, v in params.items())
        url += '?' + query
    return url

def parse_query_string(query):
    """解析查询字符串"""
    result = {}
    for pair in query.split('&'):
        if '=' in pair:
            k, v = pair.split('=', 1)
            result[k] = v
    return result

def validate_username(username):
    """验证用户名"""
    return len(username) >= 3 and username.isalnum()

def validate_password(password):
    """验证密码强度"""
    if len(password) < 8:
        return False
    has_letter = any(c.isalpha() for c in password)
    has_digit = any(c.isdigit() for c in password)
    return has_letter and has_digit

def validate_phone(phone):
    """验证手机号"""

def validate_id_card(id_card):
    """简单身份证验证"""
    return len(id_card) == 18 and id_card[:-1].isdigit()

# ==================== 位操作 ====================
def set_bit(value, bit):
    """设置位"""
    return value | (1 << bit)

def clear_bit(value, bit):
    """清除位"""
    return value & ~(1 << bit)

def is_bit_set(value, bit):
    """检查位是否设置"""
    return (value & (1 << bit)) != 0

def toggle_bit(value, bit):
    """切换位"""
    return value ^ (1 << bit)

def calculate_circle_area(radius):
    """计算圆面积"""
    return math.pi * radius ** 2

def calculate_circle_circumference(radius):
    """计算圆周长"""
    return 2 * math.pi * radius

def calculate_rectangle_area(width, height):
    """计算矩形面积"""
    return width * height

def calculate_rectangle_perimeter(width, height):
    """计算矩形周长"""
    return 2 * (width + height)

def calculate_triangle_area(a, b, c):
    """海伦公式计算三角形面积"""
    s = (a + b + c) / 2
    return math.sqrt(s * (s - a) * (s - b) * (s - c))

class DataConverter:
    """数据转换工具"""
    
    @staticmethod
    def list_to_dict(items, key_func):
        """列表转字典"""
        result = {}
        for item in items:
            key = key_func(item)
            result[key] = item
        return result
    
    @staticmethod
    def flatten_list(nested_list):
        """展平嵌套列表"""
        result = []
        for item in nested_list:
            if isinstance(item, list):
                result.extend(DataConverter.flatten_list(item))
            else:
                result.append(item)
        return result
    
    @staticmethod
    def chunk_list(items, chunk_size):
        """分块处理列表"""
        chunks = []
        for i in range(0, len(items), chunk_size):
            chunks.append(items[i:i + chunk_size])
        return chunks

class EventEmitter:
    """简单的事件发射器"""
    
    def __init__(self):
        self.listeners = {}
    
    def on(self, event_name, callback):
        """注册事件监听器"""
        if event_name not in self.listeners:
            self.listeners[event_name] = []
        self.listeners[event_name].append(callback)
    
    def emit(self, event_name, *args):
        """触发事件"""
        if event_name in self.listeners:
            for callback in self.listeners[event_name]:
                callback(*args)
    
    def off(self, event_name, callback):
        """移除事件监听器"""
        if event_name in self.listeners:
            if callback in self.listeners[event_name]:
                self.listeners[event_name].remove(callback)

# ==================== 缓存管理 ====================
class SimpleCache:
    """简单缓存实现"""
    
    def __init__(self, max_size=100):
        self.max_size = max_size
        self.cache = {}
        self.access_order = []
    
    def get(self, key):
        """获取缓存"""
        if key in self.cache:
            self.access_order.remove(key)
            self.access_order.append(key)
            return self.cache[key]
        return None
    
    def set(self, key, value):
        """设置缓存"""
        if key in self.cache:
            self.access_order.remove(key)
        elif len(self.cache) >= self.max_size:
            oldest = self.access_order.pop(0)
            del self.cache[oldest]
        self.cache[key] = value
        self.access_order.append(key)
    
    def clear(self):
        """清空缓存"""
        self.cache.clear()
        self.access_order.clear()

class Timer:
    """简单定时器"""
    
    def __init__(self):
        self.start_time = None
        self.elapsed = 0
    
    def start(self):
        """启动计时器"""
        import time
        self.start_time = time.time()
    
    def stop(self):
        """停止计时器"""
        import time
        if self.start_time:
            self.elapsed += time.time() - self.start_time
            self.start_time = None
    
    def reset(self):
        """重置计时器"""
        self.start_time = None
        self.elapsed = 0
    
    def get_elapsed(self):
        """获取经过时间"""
        import time
        if self.start_time:
            return self.elapsed + time.time() - self.start_time
        return self.elapsed

# ==================== 线程安全队列 ====================
class SafeQueue:
    """线程安全队列"""
    
    def __init__(self):
        import threading
        self.queue = []
        self.lock = threading.Lock()
    
    def put(self, item):
        """放入队列"""
        with self.lock:
            self.queue.append(item)
    
    def get(self):
        """从队列取出"""
        with self.lock:
            if self.queue:
                return self.queue.pop(0)
            return None
    
    def size(self):
        """队列大小"""
        with self.lock:
            return len(self.queue)
    
    def clear(self):
        """清空队列"""
        with self.lock:
            self.queue.clear()

# ==================== 状态机 ====================
class StateMachine:
    """简单状态机"""
    
    def __init__(self, initial_state):
        self.current_state = initial_state
        self.transitions = {}
    
    def add_transition(self, from_state, to_state, condition=None):
        """添加状态转换"""
        if from_state not in self.transitions:
            self.transitions[from_state] = []
        self.transitions[from_state].append((to_state, condition))
    
    def transition(self, event=None):
        """执行状态转换"""
        if self.current_state in self.transitions:
            for to_state, condition in self.transitions[self.current_state]:
                if condition is None or condition(event):
                    self.current_state = to_state
                    return True
        return False
    
    def get_state(self):
        """获取当前状态"""
        return self.current_state

            self.observers.remove(observer)

        """通知所有观察者"""
        for observer in self.observers:
            observer.update(self)
