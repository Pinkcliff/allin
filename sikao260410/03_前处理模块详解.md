# 前处理模块详解

## 模块概述

前处理模块是CFD (计算流体力学) 仿真的前处理器，负责设置仿真参数、生成计算网格和创建3D场景。

## 核心文件

| 文件 | 功能描述 |
|------|----------|
| `pre_processor_window.py` | 主窗口逻辑，1000+行核心代码 |
| `ui_main_window.py` | UI布局定义 |
| `scene_generator.py` | 3D场景生成器 |
| `file_handler.py` | 文件读写处理 |
| `fan_id_generator.py` | 风扇ID矩阵生成 |
| `grid_utils.py` | 网格坐标生成 |
| `pre_processor_config.py` | 配置参数 |
| `fan_curve_14600.txt` | 风扇PQ曲线数据 |

## 边界计算

```python
def update_domain_bounds():
    """根据当前参数计算边界"""
    fan_wall_x_size = FAN_ARRAY_SHAPE[1] * FAN_WIDTH
    fan_wall_y_size = FAN_ARRAY_SHAPE[0] * FAN_WIDTH

    # 根据IS_GROUNDED标志决定ymin
    ymin_val = 0.0 if IS_GROUNDED else -MARGIN_Y

    # 毫米单位的边界
    DOMAIN_BOUNDS = {
        'xmin': -MARGIN_X,
        'xmax': fan_wall_x_size + MARGIN_X,
        'ymin': ymin_val,
        'ymax': fan_wall_y_size + MARGIN_Y,
        'zmin': -INLET_LENGTH * 1000,
        'zmax': FAN_THICKNESS + OUTLET_LENGTH * 1000
    }

    # 米单位的边界
    DOMAIN_BOUNDS_M = {key: value / 1000.0 for key, value in DOMAIN_BOUNDS.items()}
```

## 风扇ID生成

风扇ID格式: `X000Y000`
- X: 列号 (000-039)
- Y: 行号 (000-039)

示例:
- X000Y000: 第0列第0行
- X020Y015: 第20列第15行
- X039Y039: 第39列第39行

## 场景生成流程

```python
class SceneGenerator:
    def generate_scene(self, config):
        # 1. 创建计算域
        domain = self.create_domain(config)

        # 2. 生成风扇阵列
        fans = self.generate_fan_array(config)

        # 3. 创建网格
        grid = self.generate_grid(config)

        # 4. 设置边界条件
        boundaries = self.setup_boundaries(config)

        # 5. 创建PyVista场景
        scene = self.create_vtk_scene(domain, fans, grid, boundaries)

        return scene
```

## 网格生成

```python
def generate_grid(config):
    """生成结构化网格"""
    # 组件网格 (风扇周围)
    component_grid = create_component_grid(
        cells=COMPONENT_GRID_CELLS,
        size=ENVIRONMENT_GRID_SIZE
    )

    # 环境网格 (外部区域)
    environment_grid = create_environment_grid(
        bounds=DOMAIN_BOUNDS,
        stretch_ratio=(STRETCH_RATIO_XY, STRETCH_RATIO_XY, STRETCH_RATIO_Z)
    )

    # 合并网格
    final_grid = merge_grids(component_grid, environment_grid)

    return final_grid
```

## 仿真控制

```python
class SimulationController:
    def run_simulation(self):
        # 1. 检查参数完整性
        if not self.validate_parameters():
            return False

        # 2. 生成网格
        grid = self.generate_mesh()

        # 3. 导出求解器文件
        self.export_solver_files()

        # 4. 启动求解器
        self.launch_solver()

        # 5. 监控残差
        self.monitor_residuals()

        return True
```

## 与风场设置的接口

```python
def receive_wind_field_data(wind_field_data):
    """
    接收来自风场设置的数据

    Args:
        wind_field_data: {
            'grid_data': np.ndarray,  # 40x40转速百分比
            'max_rpm': int,
            'time_series': [...]
        }
    """
    rpm_data = wind_field_data['rpm_data']

    # 转换为风扇参数
    for i in range(40):
        for j in range(40):
            fan_rpm = rpm_data[i, j]
            # 设置风扇转速
            set_fan_rpm(i, j, fan_rpm)
```

## 关键技术点

1. **PyVista集成** - 3D可视化的核心技术
2. **VTK管道** - 数据处理和渲染
3. **多线程处理** - 网格生成在后台线程
4. **参数验证** - 确保参数合理性
5. **文件格式转换** - 支持多种求解器格式

## 注意事项

1. 网格生成可能耗时较长，需要进度提示
2. 大量风扇数据需要优化内存使用
3. 3D场景需要适当的显卡支持
4. 求解器集成需要根据实际求解器调整
