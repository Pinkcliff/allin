# 仪表盘模块详解

## 模块概述

仪表盘模块是整个系统的全局监控中心，负责实时显示系统各部分的状态、环境参数、电力参数等信息。

## 核心文件

| 文件 | 功能描述 |
|------|----------|
| `ui_main_window.py` | 主窗口类，700+行，核心窗口逻辑 |
| `ui_docks.py` | 15个监控面板定义 |
| `ui_custom_widgets.py` | 自定义UI组件（指示器、拖拽框等） |
| `ui_chart_widget.py` | 实时图表组件 |
| `ui_motion_capture.py` | 动捕相机状态监控 |
| `core_theme_manager.py` | 深色/浅色主题管理 |
| `core_data_simulator.py` | 数据模拟器 |
| `ui_sensor_collection.py` | 传感器数据采集和查看 |
| `ui_sensor_dock.py` | 传感器数据dock |

## 监控面板

### 1. 系统面板
- 系统健康状态指示器
- 设备开关状态
- 时间显示

### 2. 通讯面板
- 各模块通讯状态
- 通讯协议类型
- 通讯响应时间
- 包含电力模块（电流、电压、功率）

支持的协议:
- TCP/IP (主控制器)
- EtherCAT (电驱、传感器)
- Modbus (伺服、执行器)
- API (动捕、仿真)

### 3. 环境面板
- 温度传感器 (100个)
- 风速传感器 (100个)
- 温湿度传感器 (4个)
- 大气压力传感器 (1个)

### 4. 日志面板
- 系统日志表格
- 活动告警表格
- 实时日志更新

### 5. 俯仰·造雨·示踪面板
组合面板，包含:
- 俯仰伺服控制
- 造雨系统控制
- 示踪剂控制

### 6. 风机面板
- 风机状态显示
- 风机控制

### 7. 动捕面板
- 20个相机状态灯
- 相机参数设置
- 打开动捕采集按钮
- 打开动捕视场按钮

### 8. 传感器数据面板
- 数据采集标签页
- 数据查看标签页
- MongoDB数据标签页

### 9. 其他面板
- 标定面板
- 仿真面板
- 训练面板
- 设置面板
- TCP/IP设置面板
- Modbus RTU设置面板
- EtherCAT设置面板
- API设置面板

## 主要功能

### 1. 设备控制
```python
# 开关机按钮
self.switch_button.clicked.connect(self.handle_switch_toggle)

# 急停按钮
self.estop_button.clicked.connect(self.handle_estop)
```

### 2. 主题切换
```python
# 深色/浅色主题切换
self.theme_switch = ThemeSwitch()
self.theme_switch.toggled.connect(self._on_theme_switched)
```

### 3. Dock管理
- 默认显示: 系统、通讯、环境、日志
- 动态显示: 其他面板
- 级联定位: 自动排列dock位置
- 最小化恢复: 保存dock状态

### 4. 动捕相机管理
```python
# 20个相机的模拟数据
for i in range(1, 21):
    self.camera_data[i] = {

        "dhcp": False,
        "firmware": "v1.2.3",
        "framerate": 210,

    }
```

## 数据模拟器

`core_data_simulator.py` 提供:
- 电流数据模拟
- 电压数据模拟
- 功率数据模拟
- 通讯速度模拟

## 传感器数据采集

`ui_sensor_collection.py` 提供:

### 数据采集功能
- 配置采集名称和时长
- 实时进度显示
- 数据预览
- 暂停/停止控制

### 数据查看功能
- 历史采集记录列表
- 数据表格查看
- 图形查看 (风扇、风速、温度)
- MongoDB同步

### 传感器配置
- 风扇PWM: 1600个 (0-1000)
- 温度传感器: 100个 (-20~80℃)
- 风速传感器: 100个 (0~30m/s)
- 温湿度传感器: 4个
- 大气压力传感器: 1个
- 采样率: 10次/秒

## 启动方式

```bash
# 通过融合系统启动器
python main.py
# 选择 "全局监控仪表盘" 或 "融合系统"

# 直接启动
python 仪表盘/main.py
```

## 关键技术点

1. **PySide6信号槽机制** - 用于组件间通信
2. **QTimer定时器** - 实时数据更新
3. **主题管理器** - 动态样式切换
4. **Dock窗口管理** - 浮动面板系统
5. **线程安全** - 采集工作线程与UI线程分离
