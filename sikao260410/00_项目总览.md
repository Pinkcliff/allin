# 微小型无人机智能风场测试评估系统 - 项目总览

## 项目概述

这是一个专业的**微小型无人机智能风场测试评估系统**，用于风洞测试环境中的数据采集、风场控制和实时监控。

## 核心功能

1. **40x40风扇阵列控制系统** - 1600个独立风扇的精密控制
2. **CFD前处理模块** - 计算流体力学仿真前处理
3. **全局监控仪表盘** - 实时系统状态监控
4. **传感器数据采集** - 多类型传感器数据采集和存储
5. **运动捕捉系统** - 动捕数据采集和处理
6. **硬件控制** - Modbus/EtherCAT协议硬件控制

## 通信协议支持

| 协议 | 用途 | 模块 |
|------|------|------|
| TCP/IP | 主控制器通信 | 主控制器 |
| EtherCAT | 电驱、风速、温度、湿度传感器 | 传感器 |
| Modbus | 俯仰伺服、造雨、喷雾、电力 | 执行器 |
| API | 动捕、训练、仿真 | 软件 |

## 主要特点

1. **模块化设计** - 各模块可独立运行
2. **工业级通信** - 支持多种工业通信协议
3. **实时数据采集** - 10Hz采样率
4. **3D可视化** - PyVista实现风扇阵列实时显示
5. **数据持久化** - Redis实时存储 + MongoDB持久化
6. **主题系统** - 深色/浅色主题切换

## 数据流

```
用户操作
    │
    ▼
风场设置 → 配置风扇转速模式
    │
    ├─→ 保存为模板
    │
    └─→ 进入仿真
         │
         ▼
前处理模块 → CFD网格生成
    │
    ▼
仿真运行 → 计算结果
    │
    ▼
仪表盘 → 实时显示
```
