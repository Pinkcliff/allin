# 硬件控制模块详解

## 模块概述

硬件控制模块负责与底层硬件设备的通信和控制，支持Modbus和EtherCAT协议，主要用于风扇阵列的控制。

## 核心文件

| 文件 | 功能描述 |
|------|----------|
| `modbus_fan.py` | Modbus风扇控制实现 |
| `config.py` | 硬件控制配置参数 |
| `fan_encoder.py` | 风扇数据编码/解码 |
| `wind_field_integration.py` | 风场数据集成 |
| `examples.py` | 使用示例 |
| `demo_logging.py` | 日志演示 |
| `batch_control.py` | 批量控制 |
| `CSV记录说明.md` | CSV格式说明 |
| `100板子控制说明.md` | 多板控制说明 |

## 架构变更 (协议重构)

### 原有架构
```
10路链路 × 10电驱板 (级联)
    │
    └─→ 令牌环转发
    └─→ 广播控制包 (8000字节)
```

### 新架构
```
100个独立电驱板 (无级联)
    │
    └─→ 一对一通信
    └─→ 单播控制包 (100字节)
```

## 通讯协议设计

### 1. 控制数据包 (中控机 → 单个电驱板)

**包大小**: 100字节

#### 协议头 (16字节)

| 偏移 | 长度 | 字段名 | 类型 | 说明 |
|------|------|--------|------|------|
| 0 | 2 | magic | char[2] | "FC" (Fan Control) |
| 2 | 1 | board_id | uint8 | 电驱板ID (0-99) |
| 3 | 2 | sequence_number | uint16 | 序列号 |
| 5 | 4 | timestamp | uint32 | 时间戳(毫秒) |
| 9 | 2 | control_flags | uint16 | 控制标志 |
| 11 | 5 | reserved | uint8[5] | 保留字段 |

#### 控制数据 (80字节)

每个风扇5字节，16个风扇:

| 偏移 | 长度 | 字段名 | 类型 | 说明 |
|------|------|--------|------|------|
| 0 | 1 | fan_id | uint8 | 风扇ID (0-15) |
| 1 | 2 | pwm_level | uint16 | PWM等级 (0-999) |
| 3 | 2 | fan_flags | uint16 | 风扇标志 |

#### 校验和 (4字节)

CRC32校验，覆盖协议头+控制数据

### 2. 状态反馈数据包 (单个电驱板 → 中控机)

**包大小**: 152字节

#### 协议头 (16字节)

| 偏移 | 长度 | 字段名 | 类型 | 说明 |
|------|------|--------|------|------|
| 0 | 2 | magic | char[2] | "FR" (Fan Response) |
| 2 | 1 | board_id | uint8 | 电驱板ID (回显) |
| 3 | 2 | sequence_number | uint16 | 序列号 (回显) |
| 5 | 4 | timestamp | uint32 | 时间戳 |
| 9 | 1 | online_status | uint8 | 在线状态 |
| 10 | 5 | reserved | uint8[5] | 保留 |

#### 状态数据 (132字节)

| 偏移 | 长度 | 字段名 | 类型 | 说明 |
|------|------|--------|------|------|
| 0 | 2 | total_current | uint16 | 总电流(mA) |
| 2 | 2 | board_voltage | uint16 | 板载电压(mV) |
| 4 | 2 | board_temp | uint16 | 板温(℃×10) |
| 6 | 32 | rpm_inlet[16] | uint16[16] | 入口级转速 |
| 38 | 32 | rpm_outlet[16] | uint16[16] | 出口级转速 |
| 70 | 16 | fan_status[16] | uint8[16] | 风扇状态 |
| 86 | 1 | online_count | uint8 | 在线数量 |
| 87 | 1 | error_count | uint8 | 故障数量 |
| 88 | 44 | reserved | uint8[44] | 保留 |

## Modbus风扇控制

```python
class ModbusFanController:
    """Modbus风扇控制器"""

    def __init__(self, config: FanConfig):
        self.config = config
        self.client = None

    def connect(self):
        """连接到Modbus设备"""
        from pymodbus.client import ModbusTcpClient

        self.client = ModbusTcpClient(
            host=self.config.device_ip,
            port=self.config.device_port,
            timeout=self.config.timeout
        )

        return self.client.connect()

    def set_single_fan_pwm(self, fan_index: int, pwm_value: int) -> bool:
        """设置单个风扇PWM"""
        if not self.config.validate_fan_index(fan_index):
            return False

        if not self.config.validate_pwm(pwm_value):
            return False

        register_addr = self.config.get_register_address(fan_index)

        try:
            result = self.client.write_register(
                address=register_addr,
                value=pwm_value,
                slave=self.config.slave_addr
            )
            return not result.isError()
        except Exception as e:
            print(f"设置风扇PWM失败: {e}")
            return False

    def set_multiple_fans_pwm(self, fan_pwm_list: List[Tuple[int, int]]) -> bool:
        """批量设置风扇PWM"""
        registers = []
        for fan_index, pwm_value in fan_pwm_list:
            if self.config.validate_fan_index(fan_index) and \
               self.config.validate_pwm(pwm_value):
                registers.append((self.config.get_register_address(fan_index), pwm_value))

        try:
            result = self.client.write_registers(
                address=registers[0][0],
                values=[r[1] for r in registers],
                slave=self.config.slave_addr
            )
            return not result.isError()
        except Exception as e:
            print(f"批量设置PWM失败: {e}")
            return False

    def read_fan_status(self, fan_index: int) -> Optional[dict]:
        """读取风扇状态"""
        register_addr = self.config.get_register_address(fan_index)

        try:
            result = self.client.read_holding_registers(
                address=register_addr,
                count=2,  # PWM + 状态
                slave=self.config.slave_addr
            )

            if not result.isError():
                return {
                    'pwm': result.registers[0],
                    'status': result.registers[1]
                }
        except Exception as e:
            print(f"读取风扇状态失败: {e}")

        return None

    def disconnect(self):
        """断开连接"""
        if self.client:
            self.client.close()
```

## 风扇数据编码

```python
class FanEncoder:
    """风扇数据编码器"""

    @staticmethod
    def encode_pwm_value(pwm: int, max_pwm: int = 1000) -> int:
        """编码PWM值"""
        # 限制范围
        pwm = max(0, min(max_pwm, pwm))
        return pwm

    @staticmethod
    def decode_pwm_value(encoded: int) -> int:
        """解码PWM值"""
        return encoded

    @staticmethod
    def encode_fan_flags(enable: bool, reverse: bool = False) -> int:
        """编码风扇标志"""
        flags = 0
        if enable:
            flags |= 0x01  # bit0: 启停
        if reverse:
            flags |= 0x02  # bit1: 反转
        return flags

    @staticmethod
    def decode_fan_flags(flags: int) -> dict:
        """解码风扇标志"""
        return {
            'enabled': bool(flags & 0x01),
            'reverse': bool(flags & 0x02)
        }
```

## 风场数据集成

```python
def wind_field_to_fan_control(wind_field_data: np.ndarray,
                               board_config: List[FanConfig]) -> List[dict]:
    """
    将风场数据转换为风扇控制指令

    Args:
        wind_field_data: 40x40风场数据 (转速百分比)
        board_config: 电驱板配置列表

    Returns:
        控制指令列表
    """
    control_commands = []

    # 遍历所有电驱板
    for board_idx, config in enumerate(board_config):
        # 计算该板对应的风扇范围
        start_row = (board_idx // 10) * 4
        start_col = (board_idx % 10) * 4

        # 收集该板的风扇数据
        fan_pwm_list = []
        for fan_idx in range(config.fan_count):
            row = start_row + (fan_idx // 4)
            col = start_col + (fan_idx % 4)

            # 获取转速百分比
            speed_percent = wind_field_data[row, col]
            # 转换为PWM值
            pwm_value = int(speed_percent / 100 * config.pwm_max)

            fan_pwm_list.append((fan_idx, pwm_value))

        # 创建控制指令
        control_commands.append({
            'board_id': board_idx,
            'board_ip': config.device_ip,
            'fan_pwm_list': fan_pwm_list
        })

    return control_commands
```

## 批量控制

```python
async def batch_control_fan_boards(control_commands: List[dict]) -> dict:
    """
    异步批量控制多个电驱板

    Args:
        control_commands: 控制指令列表

    Returns:
        控制结果 {board_id: success}
    """
    results = {}

    async def control_single_board(command: dict):
        controller = ModbusFanController(
            FanConfig(device_ip=command['board_ip'])
        )

        try:
            if controller.connect():
                success = controller.set_multiple_fans_pwm(
                    command['fan_pwm_list']
                )
                results[command['board_id']] = success
            else:
                results[command['board_id']] = False
        finally:
            controller.disconnect()

    # 并发控制所有板
    tasks = [control_single_board(cmd) for cmd in control_commands]
    await asyncio.gather(*tasks)

    return results
```

## 日志记录

```python
class FanControlLogger:
    """风扇控制日志记录器"""

    def __init__(self, log_file: str):
        self.log_file = log_file
        self.csv_file = None

    def start_logging(self):
        """开始记录"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        self.csv_file = f"fan_control_{timestamp}.csv"

        # 创建CSV文件
        with open(self.csv_file, 'w', newline='') as f:
            writer = csv.writer(f)
            writer.writerow([
                'timestamp', 'board_id', 'fan_id',
                'pwm_value', 'status'
            ])

    def log_command(self, board_id: int, fan_id: int,
                    pwm_value: int, status: str):
        """记录控制指令"""
        with open(self.csv_file, 'a', newline='') as f:
            writer = csv.writer(f)
            writer.writerow([
                datetime.now().isoformat(),
                board_id, fan_id, pwm_value, status
            ])

    def stop_logging(self):
        """停止记录"""
        self.csv_file = None
```

controller = ModbusFanController(config)

if controller.connect():
    # 设置单个风扇
    controller.set_single_fan_pwm(0, 500)

    fan_list = [(i, i*50) for i in range(16)]

## 注意事项

1. **网络配置**: 确保中控机和电驱板在同一网段
2. **通信超时**: 根据网络情况调整超时时间
3. **PWM精度**: 保持0-999的1000级精度
4. **错误处理**: 实现完善的重连机制
5. **日志记录**: 记录所有控制指令用于调试
