## 项目关键指标

| 指标 | 数值 |
|------|------|
| 代码行数 | 约15000+行 |
| 模块数量 | 7个主要模块 |
| 风扇数量 | 1600个 (40x40) |
| 传感器数量 | 1805个 |
| 支持协议 | 4种 (TCP/IP, EtherCAT, Modbus, API) |
| 采样率 | 10Hz |
| 数据库 | Redis + MongoDB |

## 核心技术栈

- **语言**: Python 3.11+
- **UI**: PySide6 (Qt6)
- **3D**: PyVista + VTK
- **图表**: PyQtGraph
- **数据库**: Redis + MongoDB
- **通信**: Modbus, EtherCAT, TCP/IP
- **数值**: NumPy, Pandas
