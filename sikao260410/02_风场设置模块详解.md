# 风场设置模块详解

## 模块概述

风场设置模块是控制40x40风扇阵列的核心模块，提供可视化编辑界面来配置风扇转速模式和时间序列。

## 核心文件

| 文件 | 功能描述 |
|------|----------|
| `main_window.py` | 主窗口逻辑，28.7KB |
| `canvas_widget.py` | 风扇阵列可视化画布，25.4KB |
| `floating_windows.py` | 工具面板浮动窗口，19.7KB |
| `timeline_widget.py` | 时间序列配置，5.6KB |
| `commands.py` | 撤销/重做命令模式 |
| `simulation_interface.py` | CFD仿真接口 |
| `template_library.py` | 模板保存/加载 |
| `properties_panel.py` | 属性面板 |
| `config.py` | 配置参数 |
| `utils.py` | 工具函数 |

## 主要功能

### 1. 风扇阵列可视化

40x40风扇阵列 (1600个风扇) 的可视化显示:
- 颜色映射: 淡蓝 → 绿 → 黄 → 红
- 网格大小: 720x720像素
- 单元格大小: 16x16像素
- 间距: 2像素

风扇ID格式: `X000Y000` (列号+行号)

### 2. 编辑工具

#### 点选模式
- 点击选择单个风扇
- Shift+点击: 多选
- Ctrl+点击: 反选

#### 笔刷模式
- 可调节笔刷大小
- 支持羽化效果
- 实时预览

#### 圆形模式
- 半径可调
- 支持羽化
- Shift/Ctrl修饰键

#### 直线模式
- Bresenham算法
- 支持羽化
- 端点选择

#### 函数模式
12种数学函数模板:
1. 高斯分布 (Gaussian)
2. 径向波 (Radial Wave)
3. 线性渐变 (Linear Gradient)
4. 径向渐变 (Radial Gradient)
5. 螺旋波 (Spiral Wave)
6. 干涉图样 (Interference)
7. 驻波 (Standing Wave)
8. 棋盘格 (Checkerboard)
9. 噪声场 (Noise)
10. 多项式曲面 (Polynomial)
11. 鞍点 (Saddle)
12. 高斯波包 (Gaussian Packet)

### 3. 时间轴控制

- 播放/暂停
- 时间滑块
- 时间设置:
  - 最大时间: 10秒
  - 时间分辨率: 0.1秒
  - 关键帧编辑

### 4. 模板库

- 保存当前配置
- 加载已保存模板
- 模板预览

## 核心算法

### Bresenham直线算法
```python
def bresenham_line(x0, y0, x1, y1):
    points = []
    dx = abs(x1 - x0)
    dy = abs(y1 - y0)
    sx = 1 if x0 < x1 else -1
    sy = 1 if y0 < y1 else -1
    err = dx - dy

    while True:
        points.append((x0, y0))
        if x0 == x1 and y0 == y1:
            break
        e2 = 2 * err
        if e2 > -dy:
            err -= dy
            x0 += sx
        if e2 < dx:
            err += dx
            y0 += sy

    return points
```

### 羽化算法
```python
def apply_feathering(source_cells, base_value, feather_value):
    """
    羽化算法:
    第1层: 100% 基础值
    第2层: (n-1)/n 基础值
    第3层: (n-2)/n 基础值
    ...
    第n层: 1/n 基础值
    """
    processed = set(source_cells)
    current_layer = list(source_cells)

    for layer in range(1, feather_value + 1):
        next_layer = set()
        layer_value = base_value * (feather_value - layer) / feather_value

        for row, col in current_layer:
            for dr, dc in [(0, 1), (0, -1), (1, 0), (-1, 0)]:
                nr, nc = row + dr, col + dc
                if 0 <= nr < grid_dim and 0 <= nc < grid_dim:
                    if (nr, nc) not in processed:
                        if layer_value > get_cell_value(nr, nc):
                            set_cell_value(nr, nc, layer_value)
                        next_layer.add((nr, nc))
                        processed.add((nr, nc))

        current_layer = list(next_layer)
```
