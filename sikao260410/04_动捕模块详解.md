# 动捕模块详解

## 模块概述

动捕模块用于采集运动捕捉系统的数据，支持实时数据接收、处理和CSV导出。

## 核心文件

| 文件 | 功能描述 |
|------|----------|
| `motion_capture_window.py` | 动捕采集窗口UI |
| `LuMoSDKPy/LuMoSDKClient.py` | SDK客户端封装 |
| `LuMoSDKPy/LusterFrameStruct_pb2.py` | Protobuf数据结构定义 |
| `LuMoSDKPy/PythonSample.py` | SDK使用示例 |
| `LuMoSDKPy/redis_storage_client.py` | Redis存储客户端 |
| `test_sdk_only.py` | SDK测试脚本 |
| `test_motion_capture.py` | 动捕采集测试 |

## 数据格式

### Protobuf结构

```protobuf
message LusterFrameStruct {
    uint32 frame_id = 1;           // 帧ID
    uint64 timestamp = 2;          // 时间戳
    uint32 camera_sync_time = 3;   // 相机同步时间
    uint32 data_broadcast_time = 4;// 数据广播时间

    repeated Marker markers = 10;   // 标记点列表
    repeated RigidBody rigid_bodies = 20; // 刚体列表
    TimeCode time_code = 30;       // 时码
}

message Marker {
    uint32 id = 1;                 // 标记点ID
    string name = 2;               // 标记点名称
    float x = 3;                   // X坐标
    float y = 4;                   // Y坐标
    float z = 5;                   // Z坐标
}

message RigidBody {
    uint32 id = 1;                 // 刚体ID
    string name = 2;               // 刚体名称
    Position position = 3;         // 位置
    Rotation rotation = 4;         // 旋转
    Velocity velocity = 5;         // 速度
    Acceleration acceleration = 6; // 加速度
    EulerAngle euler_angle = 7;    // 欧拉角
}
```

### CSV输出格式

```csv
frame_id,timestamp,camera_sync_time,data_broadcast_time,
```

### 2. 数据采集

```python
class MotionCaptureWorker(QThread):
    """数据采集工作线程"""

    def __init__(self, ip_address, save_directory):
        super().__init__()
        self.ip_address = ip_address
        self.save_directory = save_directory
        self.sdk_client = None
        self.csv_file = None
        self.data_timeout = 5.0  # 5秒无数据自动停止

    def run(self):
        # 1. 连接到动捕系统
        self.sdk_client = LuMoSDKClient(self.ip_address)

        # 2. 订阅数据流
        self.sdk_client.subscribe()

        # 3. 创建CSV文件
        self.csv_file = self.create_csv_file()

        # 4. 接收数据循环
        while self.running:
            try:
                # 接收数据 (带超时)
                data = self.sdk_client.receive_data(timeout=1.0)

                if data:
                    # 解析Protobuf数据
                    frame = self.parse_frame(data)

                    # 写入CSV
                    self.write_to_csv(frame)

                    # 发送更新信号
                    self.data_received.emit(frame)

                    # 重置超时计时器
                    self.last_data_time = time.time()

                else:
                    # 检查超时
                    if time.time() - self.last_data_time > self.data_timeout:
                        raise TimeoutError("无数据超时")

            except Exception as e:
                self.error_occurred.emit(str(e))
                break

        # 5. 清理资源
        self.cleanup()
```

### 4. CSV文件管理

```python
def create_csv_file(self):
    """创建CSV文件"""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    filename = f"LuMo_Data_{timestamp}_markers{self.marker_count}.csv"
    filepath = os.path.join(self.save_directory, filename)

    with open(filepath, 'w', newline='', encoding='utf-8') as f:
        writer = csv.writer(f)

        # 写入表头
        header = self.generate_csv_header()
        writer.writerow(header)

    return filepath

def write_to_csv(self, frame):
    """写入数据到CSV"""
    with open(self.csv_file, 'a', newline='', encoding='utf-8') as f:
        writer = csv.writer(f)
        row = self.frame_to_csv_row(frame)
        writer.writerow(row)
```

## Redis存储集成

可选的Redis存储功能:

```python
class RedisStorageClient:
    """Redis存储客户端"""

    def __init__(self, host='localhost', port=6379, db=0):
        self.redis_client = redis.Redis(host=host, port=port, db=db)
        self.key_prefix = "motion_capture:data"

    def save_frame(self, frame):
        """保存单帧数据"""
        key = f"{self.key_prefix}:{frame.frame_id}"
        data = json.dumps(frame.to_dict())
        self.redis_client.set(key, data, ex=3600)  # 1小时过期

    def get_frame(self, frame_id):
        """获取单帧数据"""
        key = f"{self.key_prefix}:{frame_id}"
        data = self.redis_client.get(key)
        if data:
            return json.loads(data)
        return None

    def save_stream(self, frames):
        """批量保存帧数据"""
        pipe = self.redis_client.pipeline()
        for frame in frames:
            key = f"{self.key_prefix}:{frame.frame_id}"
            data = json.dumps(frame.to_dict())
            pipe.set(key, data, ex=3600)
        pipe.execute()
```

## 故障排除

### 问题1: 连接失败

**检查清单:**
1. 动捕系统是否已启动
2. IP地址是否正确
3. 端口6868是否开放
4. 网络连接是否正常

### 问题2: 无数据接收

**检查清单:**
1. 动捕系统是否在广播数据
2. 标记点是否正确设置
3. 相机是否正常工作

### 问题3: Protobuf解析错误

**解决方案:**
```bash
# 重新安装protobuf
pip uninstall protobuf
pip install protobuf==3.20.0
```
