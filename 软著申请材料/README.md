## 软件功能概览

### 核心功能

| 功能模块 | 主要功能 |
|---------|---------|
| **全局监控仪表盘** | 系统资源、通讯状态、环境参数、日志中心、动捕数据、风机状态监控 |
| **风场编辑器** | 3D可视化编辑、风速区域、自定义表达式、模板库、时间轴控制 |
| **硬件控制** | 1600台风扇控制、Modbus/EtherCAT通讯、CSV序列回放 |
| **传感器数据采集** | 1800+通道采集、Redis缓存、MongoDB同步、数据分析 |
| **CFD前处理** | 场景生成、网格生成、边界条件、VTM导出 |
| **动捕数据可视化** | CSV加载、3D轨迹、播放控制、统计分析 |

### 技术特性

- **多线程架构**：UI与数据处理分离
- **信号槽机制**：模块间松耦合通信
- **双层存储**：Redis + MongoDB
- **实时3D渲染**：OpenGL高性能渲染
- **表达式解析**：动态数学表达式解析
- **多协议支持**：TCP/IP、EtherCAT、Modbus

### 性能指标

| 指标 | 数值 |
|------|------|
| 最大采集频率 | 100Hz |

## 使用说明

### 1. 申请前准备

1. **阅读材料**：先阅读软件说明书和功能清单，了解软件全貌
2. **填写申请表**：参考申请表模板，填写实际信息
3. **准备身份证明**：准备营业执照或身份证复印件

### 2. 提交申请

1. **线上申请**（推荐）：
   - 访问中国版权保护中心官网
   - 注册账号并登录
   - 填写申请信息
   - 上传申请材料

2. **线下申请**：
   - 打印所有申请材料
   - 邮寄或现场提交

### 3. 审查流程

```
提交申请 → 受理审查 → 补正通知（如有）→ 审查通过 → 领取证书
           (约3-5个工作日)    (约30个工作日)
```

### 4. 注意事项

1. 源代码文档必须包含前30页和后30页
2. 软件名称应以"软件"、"系统"、"平台"等结尾
3. 所有日期必须真实有效
4. 代码中不要包含第三方库代码
5. 删除明显的调试和测试代码
