# 微小型无人机智能风场测试评估系统
## 用户手册

---

## 一、系统简介

本手册面向《微小型无人机智能风场测试评估系统》V2.0.0 的用户，介绍系统的安装、配置和操作方法。

### 1.1 系统概述

本系统是一款专业的无人机风洞测试综合平台，集成了风场控制、数据采集、可视化监控、数据分析等功能，适用于空气动力学研究、无人机性能测试等场景。

### 1.2 主要功能

| 功能模块 | 描述 |
|---------|------|
| 全局监控仪表盘 | 统一监控各子系统状态 |
| 风场编辑器 | 可视化配置风场参数 |
| 硬件控制 | 控制风扇阵列等设备 |
| 传感器数据采集 | 采集和存储传感器数据 |
| CFD前处理 | 生成仿真前处理文件 |
| 动捕数据可视化 | 处理动作捕捉数据 |

## 二、安装与配置

### 2.1 系统要求

**硬件要求：**
- CPU: Intel i5 或同等性能处理器
- 内存: 8GB 以上（推荐 16GB）
- 硬盘: 100GB 可用空间
- 网络: 千兆以太网

**软件要求：**
- 操作系统: Windows 10/11 (64位) 或 Linux
- Python: 3.8 或更高版本
- 数据库: Redis 6.0+, MongoDB 4.4+ (可选)

### 2.2 安装步骤

1. **安装Python环境**
   ```
   下载并安装 Python 3.8+
   确保 pip 可用
   ```

2. **安装依赖库**
   ```bash
   pip install PySide6 redis pymongo matplotlib numpy pandas pyqtgraph
   ```

3. **安装数据库服务**（可选）
   ```
   安装 Redis 服务并启动
   安装 MongoDB 服务并启动（用于长期数据存储）
   ```

4. **配置系统参数**
   ```
   编辑 config.py 文件
   设置通讯参数、数据库连接等
   ```

### 2.3 目录结构

```
all/
├── main.py              # 主入口程序
├── config.py            # 配置文件
├── 仪表盘/              # 仪表盘模块
├── 风场设置/            # 风场编辑模块
├── 硬件控制/            # 硬件控制模块
├── 风场编辑器/          # 风场编辑器模块
├── 前处理/              # CFD前处理模块
├── 核心模块/            # 核心数据模块
└── 资源/                # 图片等资源文件
```

## 三、快速开始

### 3.1 启动系统

双击运行 `main.py` 或在命令行执行：
```bash
python main.py
```

系统将显示启动器界面，提供三个选项：

1. **全局监控仪表盘** - 完整的监控界面
2. **融合系统** - 仪表盘 + 传感器数据
3. **传感器数据采集** - 独立的数据采集界面

### 3.2 界面介绍

**主界面布局：**

```
+----------------------------------------------------------+
|  菜单栏: 文件 | 编辑 | 视图 | 工具 | 帮助                 |
+----------------------------------------------------------+
|  工具栏: [系统监控] [通讯] [环境] [日志] [动捕] [风机]    |
+----------------------------------------------------------+
|                                                          |
|  +----------+  +----------------------------+            |
|  |          |  |                            |            |
|  |  Dock    |  |      主显示区域            |            |
|  |  窗口    |  |                            |            |
|  |  区域    |  +----------------------------+            |
|  |          |                                        ...
|  +----------+                                            |
|                                                          |
+----------------------------------------------------------+
|  状态栏: 系统状态 | 连接状态 | 时间                      |
+----------------------------------------------------------+
```

### 3.3 基本操作

- **显示/隐藏Dock窗口**：点击工具栏按钮
- **拖拽Dock窗口**：按住标题栏拖动
- **切换标签页**：点击Dock窗口内的标签

## 四、功能模块使用说明

### 4.1 全局监控仪表盘

#### 4.1.1 系统监控

显示计算机的运行状态：
- CPU 使用率
- 内存使用情况
- 网络流量

#### 4.1.2 通讯状态

显示各通讯协议的连接状态：
- TCP/IP 连接
- EtherCAT 状态
- Modbus 设备状态
- 动捕系统连接

#### 4.1.3 环境监测

实时显示环境参数：
- 温度传感器读数（100个测点）
- 风速传感器读数（100个测点）
- 湿度传感器读数（4个测点）
- 大气压力读数

#### 4.1.4 日志中心

显示系统运行日志：
- 实时日志更新
- 日志级别过滤（INFO/WARNING/ERROR）
- 日志搜索功能
- 日志导出

### 4.2 风场编辑器

#### 4.2.1 新建风场配置

1. 点击"新建"按钮
2. 设置风场参数（名称、描述）
3. 在3D视图中添加风速区域

#### 4.2.2 添加风速区域

1. 选择"添加区域"工具
2. 在3D场景中点击确定区域位置
3. 设置区域参数：
   - 位置（X, Y, Z）
   - 尺寸（长, 宽, 高）
   - 风速表达式

#### 4.2.3 风速表达式语法

支持以下表达式：
```
# 常数风速
10

# 线性梯度
5 + x * 2

# 正弦分布
10 + 5 * sin(x)

# 复杂表达式
10 + 2*x + 3*y + sin(z*0.5)
```

#### 4.2.4 时间轴控制

1. 点击"时间轴"按钮打开时间轴窗口
2. 添加关键帧
3. 在不同关键帧设置不同的风场参数
4. 系统自动插值生成中间帧

#### 4.2.5 保存和导出

- 保存项目：保存为 `.wf` 格式
- 导出CSV：导出风速分布数据
- 导出控制序列：导出到硬件控制系统

### 4.3 硬件控制

#### 4.3.1 连接设备

1. 打开硬件控制面板
2. 配置通讯参数
3. 点击"连接"按钮

#### 4.3.2 风扇控制

**单个控制：**
- 选择单个风扇
- 设置PWM值（0-1000）
- 点击"应用"

**区域控制：**
- 选择区域（矩形/圆形）
- 设置PWM值或表达式
- 点击"应用"

**梯度控制：**
- 设置起点和终点PWM值
- 选择梯度方向
- 点击"应用"

#### 4.3.3 控制序列回放

1. 加载CSV控制序列文件
2. 点击"播放"开始回放
3. 设置回放速度

#### 4.3.4 断开连接

- 点击"断开"按钮
- 确认所有设备已停止运行

### 4.4 传感器数据采集

#### 4.4.1 创建采集任务

1. 打开传感器数据采集窗口
2. 设置采集参数：
   - 采样率（Hz）
   - 采集时长（秒）
   - 采集备注
3. 点击"开始采集"

#### 4.4.2 监控采集过程

- 实时显示采集进度
- 显示已采集样本数
- 显示预计剩余时间

#### 4.4.3 查看历史数据

1. 切换到"数据查看"标签页
2. 选择要查看的采集记录
3. 查看数据详情和统计图表

#### 4.4.4 数据导出

- 导出为CSV格式
- 同步到MongoDB数据库

### 4.5 CFD前处理

#### 4.5.1 生成场景文件

1. 加载风场配置
2. 设置CFD参数：
   - 计算域大小
   - 网格分辨率
   - 边界条件
3. 点击"生成"

#### 4.5.2 导出网格

1. 设置网格参数
2. 点击"导出网格"
3. 选择导出格式（VTM、VTK等）

### 4.6 动捕数据可视化

#### 4.6.1 加载数据

1. 点击"打开文件"
2. 选择CSV格式的动捕数据文件

#### 4.6.2 播放控制

- 播放/暂停：空格键或点击按钮
- 停止：点击停止按钮
- 跳转：拖动滑块

#### 4.6.3 数据分析

- 查看标记点轨迹
- 统计分析（最大值、最小值、平均值等）
- 直方图分布

## 五、常见问题

### 5.1 系统无法启动

**问题**：双击运行没有反应

**解决方案**：
1. 检查Python是否正确安装
2. 检查依赖库是否安装完整
3. 在命令行运行查看错误信息

### 5.2 无法连接硬件设备

**问题**：显示连接失败

**解决方案**：
1. 检查网络连接
2. 确认设备IP地址配置正确
3. 检查防火墙设置
4. 确认设备已上电

### 5.3 数据采集失败

**问题**：采集任务报错

**解决方案**：
1. 检查Redis服务是否启动
2. 确认传感器连接正常
3. 检查磁盘空间是否充足

### 5.4 3D显示异常

**问题**：3D视图显示黑屏或卡顿

**解决方案**：
1. 更新显卡驱动
2. 降低显示质量设置
3. 关闭不必要的后台程序
